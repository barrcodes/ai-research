## Introducing the Codex app
- **URL:** https://openai.com/index/introducing-the-codex-app/
- **Published:** 2026-02-02
- **Description:** Launches a new macOS desktop application for Codex that provides a centralized command center for managing and coordinating multiple AI agents in parallel. The app enables developers to supervise long-running tasks, work with agent teams across projects, and use "skills" to extend Codex capabilities beyond code generation to broader knowledge work tasks.
- **Relevance:** Directly relevant to agentic patterns and AI coding assistants. Demonstrates advanced multi-agent orchestration, tool use through skills, and showcases how agents can handle complex, long-running development tasks. This is a significant product release showing LLM capabilities applied to software development workflows.
